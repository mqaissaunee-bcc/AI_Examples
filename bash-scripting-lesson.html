What is Bash Scripting?

Bash (Bourne Again Shell) is a powerful command-line interpreter and scripting language for Unix/Linux systems. Bash scripts allow you to automate repetitive tasks, combine multiple commands, and create complex workflows.

💡 Pro Tip: Every bash script should start with a shebang line (#!/bin/bash) to specify the interpreter.

Core Concepts

Variables

Store and manipulate data using variable names. Assign values without spaces around the = sign.

username="John" filename=$3

User Input

Read user input using the read command with the -p flag for prompts.

read -p "Enter name: " user echo "Hello, $user"

Conditional Statements

Execute code based on conditions using if-then-else structures.

if [ "$EUID" -ne 0 ]; then echo "Not root" fi

Loops

Iterate over sequences or lists using for, while, and until loops.

for i in {1..5}; do echo "$i" done

Functions

Create reusable code blocks with parameters and return values.

function greet() { echo "Hello, $1" } greet "Alice"

Arrays

Store multiple values in indexed or associative arrays.

fruits=("Apple" "Orange") echo "${fruits[0]}"

File Operations

Test file existence, types, and permissions using test operators.

if [ -e "$filename" ]; then echo "File exists" fi

Command Substitution

Capture command output and use it as a variable value.

current_date=$(date) echo "Today: $current_date"

Exit Status

Check if commands succeeded or failed using $? variable.

cat file.txt echo "Status: $?"

Case Statements

Multi-way branching based on pattern matching.

case $num in 1) echo "One";; 2) echo "Two";; esac

Parameter Expansion

Manipulate variables with prefixes, suffixes, and substitutions.

filename="script.sh" echo "${filename%.sh}"

Redirections

Control input/output streams for commands and scripts.

echo "text" > output.txt find / -name file &> /dev/null

Practical Examples

Working with Variables

Variables store data that can be referenced and manipulated throughout your script.

#!/bin/bash # Assign variables name="John" age=25 city="New York" # Display variables echo "Name: $name" echo "Age: $age" echo "City: $city" # Command line arguments echo "First argument: $1" echo "All arguments: $@" echo "Number of arguments: $#"
Remember: Use double quotes around variables to prevent word splitting and handle spaces correctly.

Conditional Logic

Make decisions in your scripts based on various conditions.

#!/bin/bash # Check if user is root if [ "$EUID" -eq 0 ]; then echo "Running as root" else echo "Not running as root" exit 1 fi # Check file existence file="data.txt" if [ -e "$file" ] && [ -d "$file" ]; then echo "File exists and is a directory" elif [ -e "$file" ]; then echo "File exists" else echo "File does not exist" fi # String comparison input="yes" if [ "$input" = "yes" ]; then echo "User confirmed" fi
Test Operators: -e (exists), -f (file), -d (directory), -r (readable), -w (writable), -x (executable)

Loops and Iteration

Repeat actions using for, while, and until loops.

#!/bin/bash # For loop with range echo "Counting to 5:" for i in {1..5}; do echo "Number: $i" done # For loop with list echo "\nIterating over files:" for file in *.txt; do echo "Processing: $file" done # While loop counter=1 while [ $counter -le 3 ]; do echo "Counter: $counter" counter=$((counter + 1)) done # Until loop num=5 until [ $num -eq 0 ]; do echo "Countdown: $num" num=$((num - 1)) done

Functions

Create reusable code blocks with parameters.

#!/bin/bash # Simple function function greet() { echo "Hello, $1!" } # Function with return value function add_numbers() { local sum=$(($1 + $2)) echo $sum } # Function with multiple parameters function display_info() { echo "Name: $1" echo "Age: $2" echo "City: $3" } # Call functions greet "Alice" result=$(add_numbers 5 10) echo "Sum: $result" display_info "Bob" 30 "Boston"
Local Variables: Use 'local' keyword inside functions to avoid affecting global scope.

Arrays

Store and manipulate collections of data.

#!/bin/bash # Indexed array fruits=("Apple" "Orange" "Banana") # Access elements echo "First fruit: ${fruits[0]}" echo "All fruits: ${fruits[@]}" echo "Array length: ${#fruits[@]}" # Loop through array for fruit in "${fruits[@]}"; do echo "Fruit: $fruit" done # Associative array declare -A capitals capitals[USA]="Washington D.C." capitals[France]="Paris" capitals[Japan]="Tokyo" echo "Capital of France: ${capitals[France]}" # Loop through associative array for country in "${!capitals[@]}"; do echo "$country: ${capitals[$country]}" done

File Operations

Test file properties and perform file operations.

#!/bin/bash filename="test.txt" # Check if file exists if [ -e "$filename" ]; then echo "File exists" # Check file type if [ -f "$filename" ]; then echo "It's a regular file" elif [ -d "$filename" ]; then echo "It's a directory" fi # Check permissions if [ -r "$filename" ]; then echo "File is readable" fi if [ -w "$filename" ]; then echo "File is writable" fi if [ -x "$filename" ]; then echo "File is executable" fi else echo "File does not exist" fi # Read file line by line while IFS= read -r line; do echo "Line: $line" done < "$filename"
