① System Prompt
You are an expert curriculum designer helping community college faculty develop inclusive, engaging course materials aligned to learning outcomes.
② Persona / Role
Act as a supportive instructional coach who is direct, practical, and avoids academic jargon whenever possible.
③ Context
I'm redesigning my Intro to Psychology syllabus for fall 2026. Students are mostly first-generation, working adults in evening classes. Class meets twice a week for 75 minutes.
④ Task / Instruction
Draft three active-learning activities for the first unit on Research Methods. Each should take 10–15 minutes and require no special materials.
⑤ Examples (Few-Shot)
Good: "Think-Pair-Share: students write a hypothesis, discuss with a partner, share with class." Bad: "Watch a 20-minute documentary."
⑥ Output Format
Return each activity as: Name · Time · Materials · Step-by-step instructions · Why it works (1 sentence).
⑦ Constraints / Guardrails
Do not suggest activities requiring smartphones, internet, or prior readings. Keep language at an 8th-grade reading level.
① System Prompt
In ChatGPT, set via "Custom Instructions" (Settings) or API role:"system". GPT-4o follows it strongly; o1/o3 have reduced system prompt support.
② Persona / Role
"Act as…" framing works reliably. Custom GPTs let you bake a permanent persona into a shareable, reusable assistant.
③ Context
128K token window. Attach files (PDF, DOCX, images) or paste text. Focus degrades in very long threads — front-load key context.
④ Task / Instruction
GPT-4o handles multi-step tasks well in one prompt. o1/o3 models reason deeply — give them complex tasks with minimal hand-holding.
⑤ Examples (Few-Shot)
Highly effective. Provide as inline text or structured user/assistant turns. Dramatically improves format consistency on complex outputs.
⑥ Output Format
Use JSON mode (API) for structured data. Markdown renders in the UI. "Structured Outputs" (API) enforces an exact JSON schema.
⑦ Constraints
OpenAI's own safety filters run independently — user constraints add on top. "Do not…" instructions are generally followed by GPT-4o.
🤖
Use Custom GPTs for recurring tasks
Bundle your system prompt, persona, and instructions into a Custom GPT. Share it with colleagues — no prompt re-entry required.
🧠
GPT-4o vs. o1/o3 — choose wisely
GPT-4o is fast and great for everyday tasks. o1/o3 models are slower but excel at multi-step reasoning. Don't over-prompt the reasoning models.
📎
Attach files instead of pasting
PDFs, Word docs, spreadsheets, and images can be attached directly. This saves token space and keeps your prompt clean and focused.
