      Detailed Recommendations by Use Case
      🎯 Best overall for faculty
      Claude Sonnet 4.6 at claude.ai. The free tier allows 20–30 question generations per day. Its instruction-following is unmatched for structured JSON tasks, and it writes the most realistic enterprise scenario stems — essential for Security+ and CIS-type courses.
      💰 Best for $0 with no limits
      Gemini 1.5 Pro via Google Workspace (available free to many faculty through Google for Education). Handles massive content pastes. Validate the JSON output carefully before importing — use the Tab 3 validator.
      🏫 Best if your school has M365
      Microsoft Copilot is included with most institutional M365 licenses — no personal account or cost required. GPT-4o quality under the hood. Convenient but JSON output needs validation more often than Claude.
      🔒 Best for sensitive content
      Llama 3 via Ollama (self-hosted on your own machine). Zero data leaves your institution. Requires a reasonably modern laptop (16GB RAM). Quality is good but not as reliable as cloud models for JSON structure.
      📚 Best for advanced course design
      ChatGPT o1 or o3 (requires Plus plan at $20/mo). The reasoning models think through each question more carefully, producing more nuanced distractors and deeper scenario complexity. Worth it for graduate-level or certification exam prep.
      ⚠ What to avoid
      Avoid using Perplexity, Bing Chat in search mode, or any LLM with web browsing enabled for this task — they tend to summarize rather than generate, and their JSON output is unreliable. Disable web search before pasting the prompt.
      Tips for Getting the Best Output from Any LLM

      • Don't regenerate if it fails — fix the prompt. If the JSON comes back incomplete, add "You must return valid JSON only. Do not truncate. Return all [N] questions." at the top of the prompt.

      • Use a fresh conversation. Prior conversation context can confuse the model. Always start a new chat for each question bank generation.

      • Ask for fewer questions if JSON is truncating. Most free-tier LLMs have output token limits. 15 questions is safer than 25 for ensuring complete output.

      • If JSON is wrapped in markdown fences (```json ... ```), the Tab 3 importer strips them automatically.

      • Paste content, not file attachments for the best compatibility across all LLMs. Copy text from your slides and paste it into the "Paste Course Content" field in Tab 1.

      • Claude-specific tip: If using Claude on the free tier and hitting limits, you can split a 20-question bank into two separate 10-question prompts in separate conversations, then merge them in Tab 3.
