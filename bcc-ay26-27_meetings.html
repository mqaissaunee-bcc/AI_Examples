    › How to import into Outlook

    The download is a standard .ics file that Outlook reads natively.

    1. New Outlook / web: Calendar → Add calendar → Upload from file → choose the .ics → pick a calendar to add it to.
    2. Classic Outlook (Windows): File → Open & Export → Import/Export → Import an iCalendar (.ics) → choose Import (adds events into your calendar).
    3. Outlook for Mac: File → Import → select the .ics file.

    Times are in U.S. Eastern (local). Re-running the export later replaces nothing automatically — import only what you need.
