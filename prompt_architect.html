📡 New to SCORM? What it is & which chips to use

What is SCORM?

SCORM (Sharable Content Object Reference Model) is a technical standard that lets a web-based learning activity "talk" to your LMS. When a student finishes a SCORM-packaged activity in Canvas, Canvas automatically records their completion status and score in the gradebook — no manual grading needed.

Without SCORM, a Canvas-embedded HTML activity is just a web page: students can use it, but Canvas has no idea whether they completed it or how they scored.

Do you need SCORM?

You want Canvas to auto-record completion
You want scores in the gradebook automatically
The activity is a graded assignment or quiz
You need participation tracking by student
It's part of a module with a completion requirement
You want "passed / failed" status recorded

If it's an ungraded explore-at-your-own-pace resource, plain HTML embedded via Canvas's Redirect Tool is simpler and works fine.

Which chips to activate — three common setups

🟢
Completion tracking only (simplest) Activate SCORM 1.2 Wrapper. Canvas records "completed" when the learner reaches the end. No score is sent — just a checkmark in the gradebook.
🟡
Score + completion (most common) Activate 🏆 Score Ready + SCORM 1.2 Wrapper + 🔗 Score → SCORM Bridge. Score Ready tracks points; the Bridge wires the final score to Canvas; the Wrapper handles the LMS handshake. All three work together.
🔵
Full-featured graded activity All of the above, plus Scoring & Summary Screen (so students see their results) and optionally Progress Indicator. This gives students immediate feedback and gives Canvas a full gradebook entry.

How to deploy a SCORM activity in Canvas

After generating your HTML file, you'll need to wrap it in a SCORM package before uploading to Canvas. The free tool Reload SCORM Packager (or similar) can do this, or ask your instructional designer. Once uploaded to Canvas as an External Tool or via the SCORM player, it behaves like a native graded activity.

💡 Tip: The generated code uses a cross-origin-safe SCORM API finder, so it will preview correctly in a browser tab or an external iframe without throwing errors — it only activates the Canvas gradebook connection when it detects a real LMS environment.
