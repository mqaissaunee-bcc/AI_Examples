How to embed this in Canvas
  1. Download the .html file (or copy the code into a new file).
  2. Host it somewhere public — your GitHub Pages site works well.
  3. In Canvas, edit a Page and switch to the HTML editor (</>).
  4. Add an iframe pointing at your hosted file, e.g.
    <iframe src="https://your-site/progress.html" width="100%" height="150" style="border:0;"></iframe>
  5. Save. The bar recalculates on every page load, so it stays current with no maintenance.

Canvas strips <script> tags from pasted page content, which is why the iframe approach is necessary — the script runs inside the framed document, not the Canvas page.
