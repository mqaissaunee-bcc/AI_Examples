What is Grok?

Grok is xAI's conversational AI chatbot, developed by Elon Musk's AI company. Named after Robert Heinlein's concept of "deep understanding," Grok is designed to provide real-time, witty, and unfiltered responses with access to live data from X (formerly Twitter) and the web.

The latest and most advanced model with superior reasoning capabilities, advanced mathematics, and coding skills

Grok 3 Mini

A cost-efficient reasoning model optimized for STEM tasks requiring less world knowledge

Grok 2

Previous generation model with strong multimodal capabilities and real-time data access

Hands-On Activities

Activity 1: Real-Time Current Events Analysis (15 minutes)

Your Task: Create a current events discussion using Grok's real-time data.

Step-by-Step:

  1. Access Grok through X or grok.com
  2. Use this prompt template:
"What are the top 3 stories in [YOUR SUBJECT AREA] trending today? For each story: - Provide a clear summary - Identify different perspectives - Suggest discussion questions for college students - Include reliable sources for further reading Format this as a discussion guide for my [COURSE NAME] class."
  1. Ask follow-up questions to refine the content
  2. Request specific examples relevant to your student population

Activity 2: Think Mode Problem Solving (20 minutes)

Your Task: Use Grok's advanced reasoning for complex problem solving.

Steps:

  1. Access Grok 3 with Think mode (Premium+ required)
  2. Click the "Think" button before submitting your prompt
  3. Try these complex problems:
Mathematics: "Solve and explain: A company's profit function is P(x) = -2x¬≤ + 120x - 1000. Find the maximum profit and the production level needed." Science: "Explain the greenhouse effect using current atmospheric CO2 data and predict trends for the next decade." Coding: "Debug this Python function and explain each error: [paste your code]"
  1. Review Grok's step-by-step reasoning process
  2. Use the transparent thinking for teaching examples
