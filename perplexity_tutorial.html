What is Perplexity AI?

Perplexity AI is an AI-powered answer engine that combines real-time web search with conversational AI to provide comprehensive, cited responses. Unlike traditional search engines, Perplexity synthesizes information from multiple sources and presents it as coherent answers with clickable citations.

Hands-On Activities

Activity 1: Research Methodology Training (25 minutes)

Your Task: Learn to use Perplexity for reliable academic research.

Step-by-Step Research Process:

  1. Start with a broad research question about your subject area
  2. Use Academic focus mode for scholarly sources
  3. Analyze the quality of citations provided
  4. Follow source links to verify information
  5. Ask follow-up questions to deepen understanding
Example Research Progression: Initial Query: "What are effective strategies for improving student retention in community colleges?" Follow-up: "What specific interventions have shown measurable success?" Verification: "Can you provide peer-reviewed studies on [specific intervention]?" Application: "How would these strategies apply to rural community colleges?"

Activity 2: Creating Educational Spaces (20 minutes)

Your Task: Set up a collaborative learning environment using Spaces.

Space Creation Process:

  1. Create a new Space for a specific course or project
  2. Set up focused research areas within the Space
  3. Upload relevant course documents for analysis
  4. Invite colleagues or students to collaborate
  5. Test real-time research and discussion features
Space Organization Ideas: - Course Name: "Biology 101 Research Hub" - Sections: "Current Research," "Lab Resources," "Discussion Topics" - Uploaded Files: Syllabus, recent studies, lab protocols - Collaborators: Co-instructors, teaching assistants

Activity 3: File Analysis and Feedback (15 minutes)

Your Task: Use Perplexity's file analysis for educational documents.

Document Analysis Process:

  1. Upload a sample student paper or educational document
  2. Ask for structural analysis and improvement suggestions
  3. Request fact-checking of claims and citations
  4. Generate discussion questions based on the content
  5. Create follow-up research recommendations
Sample Analysis Prompts: "Analyze this student essay for argument structure and evidence quality." "What current research contradicts or supports the claims in this document?" "Generate 5 discussion questions that would help students think critically about this topic."

Activity 4: Model Comparison for Educational Tasks (20 minutes)

Your Task: Compare different AI models for various educational applications.

Comparison Framework:

  1. Ask the same educational question using different models
  2. Compare response styles and depth
  3. Evaluate citation quality and source diversity
  4. Assess which model works best for different task types
Task Types to Test: - Creative writing prompts (Claude vs GPT-4) - Data analysis questions (GPT-4 vs Sonar) - Current events research (All models) - Technical explanations (Compare all available)
