Checking Firewall Status

iptables -L -v -n # List all iptables rules with details
iptables -S # Show iptables rules in command format
firewall-cmd --state # Check if firewalld is running
ufw status verbose # Show detailed UFW firewall rules

Enabling & Disabling Firewalls

systemctl enable firewalld # Enable firewalld at boot
systemctl start firewalld # Start firewalld service
ufw enable # Enable UFW firewall
ufw disable # Disable UFW firewall

Allowing & Denying Traffic

iptables -A INPUT -p tcp --dport 22 -j ACCEPT # Allow SSH traffic on port 22
iptables -A INPUT -s 192.168.1.100 -j DROP # Block traffic from specific IP
firewall-cmd --permanent --add-service=https # Allow HTTPS traffic permanently

Port Forwarding & NAT

iptables -t nat -A PREROUTING -p tcp --dport 8080 -j REDIRECT --to-port 80 # Redirect port 8080 to 80
iptables -A FORWARD -i eth0 -o eth1 -j ACCEPT # Allow forwarding between interfaces

Clearing & Resetting Firewall Rules

iptables -F # Flush all iptables rules
ufw reset # Reset UFW firewall rules to default
firewall-cmd --complete-reload # Reload firewalld completely
