BRICKSTORM Malware

PRC State-Sponsored Cyber Attack on Government & IT Sectors

Joint Advisory from CISA, NSA, and Canadian Cyber Centre – December 4, 2025

Threat Overview

PRC state-sponsored cyber threat actors are conducting ongoing intrusions using BRICKSTORM malware for long-term persistence on victim systems.

Target Sectors: Government Services and Facilities, Information Technology

BRICKSTORM is a sophisticated backdoor for VMware vSphere and Windows environments. It enables stealthy access with capabilities for initiation, persistence, and secure C2 communication.

Key Malware Capabilities

Advanced Evasion Techniques

  • Multiple encryption layers: HTTPS , WebSockets , and nested TLS
  • DNS-over-HTTPS (DoH) : Conceals communications using legitimate public DNS resolvers
  • SOCKS proxy : Facilitates lateral movement and tunneling within victim networks

Persistence Mechanisms

  • Self-monitoring function: Automatically reinstalls or restarts if disrupted
  • Init file modification: Executes during system bootup
  • PATH hijacking: Ensures malware executes when legitimate commands are run

Remote Control Capabilities

  • Interactive shell access on compromised systems
  • File system browsing, upload, download, and manipulation
  • Virtual machine snapshot theft for credential extraction
  • Creation of hidden rogue VMs to evade detection

Recommended Mitigations

  • Scan for BRICKSTORM using YARA and Sigma rules from the joint MAR
  • Block unauthorized DoH providers and external DoH network traffic
  • Take inventory of all network edge devices and monitor for suspicious connectivity
  • Ensure proper network segmentation that restricts traffic from DMZ to internal network
  • Disable RDP and SMB from DMZ to internal network
  • Apply principle of least privilege for service accounts
  • Upgrade VMware vSphere servers to the latest version
  • Harden VMware vSphere environments following VMware guidance
