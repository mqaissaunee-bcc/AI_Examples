2. Configuration Simulator

Apply the configuration to Router A (NY) using the terminal below. Type the commands exactly as shown.

Phase 1: ISAKMP Policy

  1. enable
  2. configure terminal
  3. crypto isakmp policy 10
  4. encryption aes
  5. authentication pre-share
  6. exit

Phase 2: Transform Set & Crypto Map

  1. crypto ipsec transform-set MYSET esp-aes
  2. crypto map MYMAP 10 ipsec-isakmp
  3. set peer 203.0.113.2
  4. set transform-set MYSET
  5. exit
