What is Prompt Architect?

A 2-minute overview before you begin

~2 min

Prompt Architect is a form-based tool that helps you write detailed, structured prompts for AI tools like Claude, ChatGPT, or Gemini. Instead of writing a prompt from scratch, you fill in a few fields — topic, audience, interactive type — and the tool assembles a prompt that instructs the AI to build a complete, deployable HTML learning activity for your students.

You don't need coding experience. You're describing what you want; the AI does the building. The result is a single HTML file you can embed directly in Canvas, share as a link, or use as a standalone activity.

Set Your Context

Topic, Discipline, and Audience — the foundation of your prompt

~3 min

The first three fields tell the AI what you're teaching, where it fits in your curriculum, and who the students are. All three are required — they're the backbone of everything the AI generates.

Topic / Subject Matter — Be specific. "The Water Cycle" generates a better activity than "Earth Science." Include a subtopic if you have one.

Discipline / Course Area — Start typing and select from the dropdown, or type your own. This shapes the vocabulary, examples, and framing the AI uses.

Target Audience — Add one or more tags. You can combine them: "First-year nursing students" + "English language learners" will produce an activity tuned to both groups simultaneously.

Choose Your Interactive Type

Quick Combos and the type selector

~3 min

This is where you tell the AI what kind of activity to build. There are over 40 types organized into categories — quizzes, simulations, timelines, writing scaffolds, role-play scenarios, data narratives, and more.

You can select more than one type. The first selection becomes the primary — the core structure of the activity. Additional types become secondary enrichment layers. Click ★ Primary on any tag to reassign.

Quick Combos are preset bundles that pre-fill the type field with a sensible primary + secondary combination. They're the fastest way to start — click one, and the type selector is already configured for a common faculty workflow.

Set Scope & Tone

Two sliders that calibrate the feel and length of your activity

~2 min

Two sliders let you calibrate how the activity feels and how long it runs. Both are visible in Simple and Full mode.

Instructional Tone ranges from Formal to Conversational. A biology lecture might stay Balanced; an orientation activity might lean Conversational. The AI adjusts vocabulary, prompts, and feedback language accordingly.

Content Scope & Depth controls breadth and estimated time-on-task. Use this to match your class period — a "10–15 min warmup" setting produces something meaningfully different from a "45–60 min full lesson."

Fine-Tune with Chips

Full Mode's individual controls

~4 min

Switching to Full Mode reveals five categories of individual chips — toggle-able settings that add specific instructions to your prompt. Chips let you go beyond what bundles offer, mixing and matching any combination.

Reference Materials

Attach your own slides, readings, or rubrics as source material

~3 min

The Reference Materials section lets you tell the AI that you'll be attaching files when you paste the prompt. Activating the 📎 Include uploaded reference file(s) chip reveals a second set of chips for selecting what kinds of files you'll attach.

The prompt will instruct the AI to use your uploaded document as its primary content source — pulling facts, examples, terminology, and structure from your own materials rather than generating from scratch.

You can select multiple file types if you're uploading a combination — for example, a slide deck and a rubric. The prompt will acknowledge both.

Include Video

The two-step workflow for verified YouTube embeds

~4 min

Activating the 🎬 Include Video Quick Option (or the Flipped Classroom bundle) reveals the Multimedia Settings panel. This is a two-step workflow: first you configure your video preferences and generate a search prompt, then you verify the YouTube IDs before copying your final prompt.

The reason for the two-step process: AI tools can't always be trusted to find real, working YouTube videos on their own. Some LLMs hallucinate video IDs that look valid but lead nowhere. This workflow routes around that problem by separating the search from the generation.

Why verify IDs? A generated activity that embeds a broken video is a dead end for students. The verification step takes about 2 minutes and prevents that entirely — the ID Checker uses YouTube's public API to confirm each video exists before it goes into your prompt.

🔍 Generate Video Search Prompt

The three-step workflow:

  1. Configure & generate — Set count, length, and quality signals, then click Generate Video Search Prompt. A modal opens with a focused prompt tailored to your topic and preferences. Copy it.
  2. Search with Perplexity or Gemini — Both have live YouTube search and return real, verified IDs. Paste the full response back into the modal — IDs are extracted automatically.
  3. Verify with the ID Checker — Click Send to ID Checker. You'll see each video's title, thumbnail, and a link to confirm before committing. Click Use These IDs to embed them directly in your final prompt.

Copy & Use Your Prompt

From the tool to a deployable HTML activity

~3 min

When you're ready, click Copy Prompt at the bottom of the tool. The full prompt is copied to your clipboard. Then:

  1. Open your LLM of choice (Claude, ChatGPT, Gemini, etc.) in a new chat.
  2. Paste the prompt.
  3. If you activated Reference Materials, attach your files before hitting send.
  4. Let the AI generate — it will produce a complete HTML file.
  5. Copy the generated code and save it as a .html file.
  6. Embed in Canvas via the Redirect Tool, host on GitHub Pages, or share as a standalone file.

Common Workflows

Four faculty scenarios with recommended settings

~4 min

Four complete workflows showing recommended settings from start to finish for common faculty scenarios.

🖥️ Convert slides into an interactive

You have an existing PowerPoint and want students to engage with the content, not just read through slides.

  1. Quick Combo: 🖥️ Slides to Interactive
  2. Set Scope to match your class period
  3. Reference Materials → Slide deck / PowerPoint
  4. Quick Option: Student-Ready
  5. Copy prompt, paste into AI, attach .pptx

🛡️ Build an AI-resistant assignment

You want students to engage genuinely — not shortcut the activity with ChatGPT.

  1. Choose a 🛡️ high-resistance activity type
  2. Quick Option: AI-Resistant
  3. Add: Writing Workshop if writing-focused
  4. Add Reference Materials if you have course readings
  5. Pair with the AI Resistance Layer tool

♿ Ensure accessibility

You need to meet WCAG standards for students using assistive technology.

  1. Quick Option: Accessible (all 4 chips)
  2. Or Full Mode → individual accessibility chips
  3. Always include Screen Reader + High Contrast
  4. Add Reduced Motion for vestibular disorders

🎬 Design a flipped lesson

Students watch a video and complete a structured activity before coming to class.

  1. Quick Combo: 🎬 Watch & Reflect
  2. Quick Option: Flipped Classroom
  3. Quick Option: Student-Ready
  4. Set Scope to match pre-class time
  5. Use the Video Search Prompt generator

📘 Build an interactive textbook chapter

A chapter-length resource with tabbed sections, annotated visuals, checkpoints, and a closing quiz — to replace or supplement a textbook.

  1. Quick Combo: 📘 Textbook Chapter
  2. Set Scope: Full Lesson or Extended Module
  3. Quick Options: Student-Ready + Accessible + Polished Output
  4. Reference Materials → upload slides, reading, or outline
  5. Copy prompt, paste into AI, attach files
