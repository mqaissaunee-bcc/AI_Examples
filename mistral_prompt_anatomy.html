① System Prompt
You are an expert curriculum designer helping community college faculty develop inclusive, engaging course materials aligned to learning outcomes.
② Persona / Role
Act as a supportive instructional coach who is direct, practical, and avoids academic jargon whenever possible.
③ Context
I'm redesigning my Intro to Psychology syllabus for fall 2026. Students are mostly first-generation, working adults in evening classes. Class meets twice a week for 75 minutes.
④ Task / Instruction
Draft three active-learning activities for the first unit on Research Methods. Each should take 10–15 minutes and require no special materials.
⑤ Examples (Few-Shot)
Good: "Think-Pair-Share: students write a hypothesis, discuss with a partner, share with class." Bad: "Watch a 20-minute documentary."
⑥ Output Format
Return each activity as: Name · Time · Materials · Step-by-step instructions · Why it works (1 sentence).
⑦ Constraints / Guardrails
Do not suggest activities requiring smartphones, internet, or prior readings. Keep language at an 8th-grade reading level.
① System Prompt
API: role:"system" message. Le Chat uses "Custom Instructions." Mistral is known for following system prompts very faithfully — one of its key strengths.
② Persona / Role
"Act as…" framing works in system prompt or user turn. Mistral Nemo is particularly good at maintaining personas across long conversations.
③ Context
128K context window. Mistral is recognized for strong precision when retrieving specific information from long documents — less "needle-in-haystack" drift.
④ Task / Instruction
Excels at precise, well-scoped instructions. Strong function-calling support in the API. Clear tasks with defined scope consistently yield best results.
⑤ Examples (Few-Shot)
Sample-efficient — often 1–2 examples suffice. Works especially well for structured output generation tasks and maintaining consistent response templates.
⑥ Output Format
JSON mode supported in API. One of the stronger models for format compliance. Function calling enforces precise structure when used via API.
⑦ Constraints
Lighter safety defaults than OpenAI/Google — more control but more responsibility. Negative constraints are followed well. Explicit guardrails are recommended.
🎯
Precision is Mistral's strength
Mistral models are built for precise instruction-following. The more specific your prompt scope, the better. Vague prompts waste its capabilities.
🌍
Excellent multilingual support
Based in France, Mistral has first-class support for French, Spanish, Italian, and German. Ideal for world language courses or multilingual student populations.
💻
Mistral Nemo runs locally
Mistral Nemo (12B) is small enough to run on a modern laptop via LM Studio. It rivals much larger models on instruction-following and creative tasks.
